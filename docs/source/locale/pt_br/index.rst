Bem vindo ao Hyperledger Fabric
=============================

O Hyperledger Fabric é uma plataforma para soluções de ledger distribuídas,
por uma arquitetura modular que oferece altos níveis de confidencialidade, resiliência,
flexibilidade e escalabilidade. Ele é projetado para suportar implementações conectáveis
de diferentes componentes e acomodar a complexidade e os meandros que existem
através do ecossistema enxuto.

O Hyperledger Fabric oferece uma arquitetura Flexivel e extensível exclusiva
distinguindo de soluções alternativas de blockchain. Planejamento para o
O futuro do blockchain corporativo requer a construção de uma 
arquitetura de código aberto. Hyperledger Fabric é o seu ponto de partida.

É recomendado para usuários iniciantes começarem a passar por
:doc:`getting_started` seção para ganhar familiaridade com o Hyperledger Fabric
componentes e o fluxo de transação básica.  Uma vez confortável, continue
Explorando a biblioteca para demonstrações, especificações técnicas, APIs, etc.

.. note:: Se você tiver dúvidas não abordadas por esta documentação, ou se deparar com
           problemas com qualquer um dos tutoriais, visite o :doc:`questions`
          página para algumas dicas sobre onde encontrar ajuda adicional.
